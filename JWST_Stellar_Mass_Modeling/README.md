JWST Stellar Mass Modeling

What this is  
This project analyzes galaxy stellar mass using James Webb Space Telescope observational data.

Why it matters  
The work demonstrates end to end data analysis on real scientific data, including preprocessing, exploratory analysis, and interpretation of results rather than toy examples.
